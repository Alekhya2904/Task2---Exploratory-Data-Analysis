# TITLE EXPLORATORY DATA ANALYSIS
## Objective
Perform Exploratory Data Analysis on the Titanic dataset.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
## Analysis Performed
- Summary Statistics
- Missing Value Analysis
- Histogram
- Boxplot
- Correlation Heatmap
- Pairplot
## Key Findings
- Most passengers were aged 20–40 years.
- Age distribution is slightly right-skewed.
- Some elderly passengers appear as outliers.
- Feature relationships were analyzed using correlation and pairplots.
